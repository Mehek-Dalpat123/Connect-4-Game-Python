# 🎮 Connect 4 Game (Python)

A classic Connect 4 game built using **Python** and **Pygame**.

## 📌 Features

- Two-player gameplay
- Interactive graphical interface
- Win detection
- Board reset after each game
- Simple and clean design

## 🛠️ Technologies Used

- Python 3
- Pygame

## 🚀 Installation

Clone this repository:

```bash
git clone https://github.com/Mehek-Dalpat123/Connect-4-Game-Python.git
```

Install Pygame:

```bash
pip install pygame
```

Run the game:

```bash
python main.py
```
